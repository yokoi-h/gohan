Benchmarking Tips¶

We can use boom for benchmarking gohan. https://github.com/rakyll/boom

How to install¶

go get github.com/rakyll/boom

Sample Server Configuraion¶

gohan server --config-file server/server_test_mysql_config.yaml

Note: this benchmark result get done in my local laptop where benchmark client, server and database working. We need proper benchmark result in doc in future.
